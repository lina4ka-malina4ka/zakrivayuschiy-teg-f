# Закрывающий тег

Учебный проект (Яндекс Практикум): одностраничный сайт‑дневник про спринты с карточками, лайками и модальным окном «Сохранить на память».

## Что реализовано

- Адаптивная вёрстка и «резиновая» типографика/отступы
- 8 карточек с изображениями и разными CSS‑фильтрами (в т.ч. множественным)
- Кнопка Like с анимацией (сердце) и состоянием `is-liked`
- Кнопка «Сохранить на память», открывающая `<dialog>` (модалка) + стилизация `::backdrop`
- Лейблы на изображениях с `mix-blend-mode` и фолбэком для браузеров без `text-stroke`

## Технологии

- HTML5 (семантическая разметка)
- CSS: переменные, `clamp()`, `@supports`, `mix-blend-mode`, `filter`, анимации `@keyframes`
- JS (подключён скрипт лайка)

## Критерии качества (самопроверка)

- [ ] Семантика: `header`, `main`, `footer`, корректная вложенность тегов
- [ ] БЭМ‑нейминг классов
- [ ] Все изображения с `alt`
- [ ] Адаптивность в диапазонах 320–1440px
- [ ] Анимации вынесены в отдельный файл
- [ ] Используются CSS‑переменные для цветов/шрифтов
- [ ] В проекте есть 8 карточек и разные фильтры изображений
